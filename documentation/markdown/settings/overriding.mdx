To override a default value in Obsidian.css specify in a `:root` declaration after the `@import` statements. For more information about CSS variables checkout [_Using CSS variables_](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_variables) from MDN and the [this playground](https://preset-env.cssdb.org/playground).

```css
@import 'obsidian.css';

:root {
  --ff-sans-serif: 'Proxima Nova', 'Avenir Next', 'Helvetica Neue', sans-serif;
}
```
